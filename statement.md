Scope of the Project:
The scope of the Mood Tracker project includes-
A Python-based console application;
Logging daily mood values;
Viewing complete mood history;
Storing data using CSV file handling;
Optional mood trend visualization using graphs;
Simple, user-friendly interface suitable for beginners;
The project does not include advanced analytics, AI predictions, or cloud storage—these may be added in future enhancements.

Target Users:
This project is designed for-
Students;
Young adults;
Individuals wanting to improve mental and emotional awareness;
Anyone who wishes to track their mood regularly.

High-Level Features:
Mood Logging- Add daily mood with date and mood rating (1–10);
Mood History- Display previously recorded entries;
Data Storage- Automatically saves data in a CSV file.
Graph Visualization - Shows a line graph of mood trends over time;
Menu-Based Interface- Easy-to-understand console navigation.
